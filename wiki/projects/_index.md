# Projects

This category contains documentation for active repositories.

## What belongs here

- Project overviews
- Architecture decisions
- Key file locations
- Common tasks and commands
- Deployment notes
- Known issues and workarounds

## Adding new project docs

Create a new `.md` file in this directory for each project you want to document.
Use the project name as the filename (e.g., `searchbard.md`).
