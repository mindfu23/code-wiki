# Utilities

This category contains standalone helper functions and utilities.

## What belongs here

- Date/time helpers
- String manipulation utilities
- Array/object utilities
- Validation functions
- Formatting utilities
- Type guards and type utilities
- Math/calculation helpers

## Adding new utilities

Create a new `.md` file in this directory following the document template in AGENTS.md.
