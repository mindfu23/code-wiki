# Integrations

This category contains third-party API connectors and service integrations.

## What belongs here

- API client configurations
- Authentication setups (OAuth, API keys)
- SDK wrappers
- Webhook handlers
- Service connectors (databases, storage, etc.)

## Example services

- GitHub API
- OpenAI/Anthropic APIs
- Supabase
- Firebase
- Stripe
- n8n workflows

## Adding new integrations

Create a new `.md` file in this directory following the document template in AGENTS.md.
